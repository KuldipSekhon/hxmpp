HXMPP is an open source haXe library for building XMPP based clients and components,
targetting flash9+, javascript, neko, php and c++ (beta).

Source code.

Source code can be cloned from our GIT respository: 
git clone git://83.64.208.21/hxmpp.git
Or browsed online.

Installation/Usage.

Refer to the sample and test source code.
For use with the nekoVM you have to place /ndll/[YOUR-OS]/sha1.ndll somewhere reachable.

Known compiler flags:

  • XMPP_DEBUG
    Prints the XML input/output stream.
    • On flash/js firebug gets used if detected.
    • On terminal targets output gets colored.
  • JABBER_DEBUG
    Verbose debug messages.
  • JABBER_SOCKETBRIDGE
    Activate the flash socketbridge.
